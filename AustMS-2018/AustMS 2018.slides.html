We want to predict and control complex energetic systems!¶

For our purposes a dynamic system is complex when:

  • It is made up of many moving parts, or subsystems (High Dimensional).
  • The subsystems are not all of the same type (Heterogenenous).
  • The subsystems are complicated (Nonlinear and/or Noisy).
  • There are well defined boundaries between the subsystems (Network Topology).

By energetic system, we mean that energy is the quantity that is exchanged between subsystems.

BondGraphTools ¶

BondGraphTools (https://github.com/BondGraphTools) a framework for modelling energetic systems.

  • Based upon the bond graph and port-Hamiltonian modelling.
  • Provies a minimal object-oriented interface for constructing, composiong and simulating models.
  • Implemented in python using the scipy stack.
  • Computational 'grunt' is provided via julia and DifferentialEquations.jl.

'Bond Graphs' are a multi-domain port-based graphical modelling technique used predominatly in mechantronics.
Port-Hamiltonian systems integrate geometric approaches from classical mechanics and control theory with port based modelling.

Both are concerned with the energetic systems.

  • Need a sufficiently abstract definition of a 'system'

Definition (Energetic System)¶

An energetic system is a tuple $(M, \mathcal{D}, U,\Phi)$ where the

  • state space $M$ is a differentiable manifold of $\dim(M) = m\ge 0$ with tangent bundle $TM$.
  • port space $\mathcal{D} \subset \mathcal{E} \times \mathcal{F}$ is a Dirac structure where $\mathcal{F}$ is a linear space, $\mathcal{E} = \mathcal{F}^*$ and $\dim{\mathcal{D}} = \dim{\mathcal{F}}= n \ge 0$.
  • control space $U \subset C^r:\mathbb{R}_+ \rightarrow \mathbb{R^k}$ with $k\ge 0$ are sufficiently differentiable control functions.
  • consitutive relation is a smooth map $\Phi: TM \times \mathcal{D} \times U\times\mathbb{R}_+ \rightarrow \mathbb{R}^{m+n}$ with $m+n >0$ such that the system dynamics satisfy the differential-algebraic equations $$\Phi(x',x,e,f,u,t)=0.$$

The incoming power (instantaenous rate of energy change, positive corresponding to inwards by convention) is given by $P_\text{in} = \left<e,f\right>$ for $(e,f)\in \mathcal{D}$

  • Energetic system is a 'Dynamic System' in the traditional sense if e,f are empty. CR is a evolution rule
  • generalisation of Port-Hamiltonians, which itself is a generalisation of 'Bond Graph' modelling, see [8].
  • port space -> effort amnd flow, much like voltage/current, force/velocity analogies.
  • control space differentiability depends on 'index' of DAE
  • CR in a sense defines a component

Composition, Encapsulation¶

 

Corollary (Composition) [8] ¶

If $\Phi_1,\Phi_2$ are energetic systems, then $\Phi_0 = (\Phi_1,\Phi_2)$ is also an energetic system.

For two ports $i,j$ are connected; then $P_i + P_j = 0$. By convetion we take this to mean that $$e_i - e_j =0\qquad f_i+f_j=0$$

 

This allows for heirarchical model building and hence computational modularity even when the system is highly interconnected.

from BondGraphTools import new, add, connect, expose

def make_rl_model():
    phi_0 = new(name="Model")
    phi_1 = new("1") # Conservation Law.
    phi_2 = new("R", value=0.3) # Dissipation.
    phi_3 = new("I", value=1) # Storage.
    port = new("SS") # Internal Port

    add(phi_0, phi_1, phi_2, phi_3, port) 

    expose(port)

    connect(port, phi_1)
    connect(phi_1, phi_2)
    connect(phi_1, phi_3)
    return phi_0

model = make_rl_model()
model.constitutive_relations
